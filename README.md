# Currency Converter

A simple and efficient web application to convert currencies using real-time exchange rate data.

## Features

- **Real-time Conversion**: Fetches the latest exchange rates to provide accurate conversions.
- **Dynamic Flag Updates**: Automatically updates the country flag based on the selected currency.
- **User-Friendly Interface**: Clean and responsive design for easy usage.

## Technologies Used

- **HTML5**: For the structure of the application.
- **CSS3**: For styling and layout.
- **JavaScript**: For fetching API data and handling DOM manipulations.
- **API**: [ExchangeRate-API](https://www.exchangerate-api.com/) for currency data.
- **Flags API**: [FlagsAPI](https://flagsapi.com/) for country flags.

## How to Run

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. Enter an amount, select the currencies, and click "Get Exchange Rate".
